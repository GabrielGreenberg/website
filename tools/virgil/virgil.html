VIRGIL

Visual LaTeX editor for academic writing.
Designed for AI compatibility.

Created by Gabriel Greenberg.   Version 0.1.17   GitHub

Install Virgil from Chrome.

Write.

  1. Start LaTeX project in local folder.Overleaf users: sync locally with Dropbox.
  2. Install Virgil.Requires Chrome.
  3. Use Virgil for writing.
  4. Use Overleaf for typesetting.Or any other LaTeX compiler.

Edit with AI.

  1. Add Virgil Plugin to Claude Cowork.Or Claude Code, or your favorite agent AI.
  2. Open project folder in Cowork.
  3. Ask Cowork for assistance with editing, bibliography, and background research.
  4. Review and approve changes in Virgil.
